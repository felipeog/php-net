# Notes Mini-Project

- Notes mini-project using **PHP** and **MySQL**
- Designs initial **database tables** and reviews **MySQL indexes**
- **Notes list** page and **individual note** pages
- Introduces **authorization**, **HTTP status codes**, and **security pitfalls**
- Explains **HTML forms** and **GET vs POST** request methods
- Persists new notes while **escaping untrusted user input**
- Covers client-side and server-side **form validation**
- Validation logic with **`Validator` class**
