[_Object-Oriented Principles in PHP_](https://laracasts.com/series/object-oriented-principles-in-php-2024-edition)

# Episodes

- Introduces **object-oriented programming** and how it differs from procedural code
- Explains **classes** as blueprints and **objects** as their instances
- Covers **constructors** and managing object state
- Introduces **DTOs (Data Transfer Objects)** with strong typing and static analysis
- Explains **dependencies, coupling, and interfaces** for flexible design
- Covers **inheritance** and **abstract classes** with practical examples
- Demonstrates **interfaces as contracts and feature filters**
- Explains **encapsulation** and visibility for safer public APIs
- Explores **getters, setters, and PHP 8.4 property hooks**
- Introduces **object composition and abstractions** for scalable systems
- Concludes with a **hands-on workshop** building interchangeable storage implementations
