# The Basics

- Compares **JavaScript and PHP** syntax, structure, and basic scope rules
- Explains JavaScript’s **loosely typed data model** and built-in types
- Covers **JavaScript operators** and how they differ from PHP’s
- Reviews **control flow**, including conditionals and multiple loop types
- Introduces **functions as first-class citizens** and different ways to define them
- Explains **variable declarations and scope** (`var`, `let`, `const`)
- Covers the **basics of JavaScript arrays**, including creation and manipulation
